KU EECS 448 TEAM 9 PROJECT README

Install XAMPP v3.2.1
	Install components for Apache, MySQL, PHPMyAdmin
		If Apache server will not start, changed port in config apache tab to 81. or end httpd.exe process that isn't functioning in task manager.

Navigate to localhost/phpmyadmin
	Create a new database entitled "eventvisualizer"
	Execute the included "databasesql.txt" in the SQL tab of the new database.

Place "eventvisualizer" folder into the htdocs folder, which is located in your xampp installion folder

Navigate to localhost/eventvisualizer/login_form.html

After this, the program should operate the same as it did during the in-class demonstration.
